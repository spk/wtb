wtb(1)
=======

NAME
----
wtb - print currently open bug ID

SYNOPSIS
--------
*wtb* [*-v* | *--version*] [*-h* | *--help*]

DESCRIPTION
-----------
what the bug - print currently open bug ID on your X

OPTIONS
-------
*-v*, *--version*::
    Show version.
*-h*, *--help*::
    Show help message.
